# Predicting Heart Disease - Kaggle Playground S6E2

This project trains a LightGBM model using Stratified K-Fold cross validation to predict the probability of heart disease.

## Project Structure
- `data/` : Kaggle dataset (not uploaded to GitHub)
- `src/` : training scripts
- `outputs/` : submission file
